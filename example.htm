Usage:

HTML Markup

<div class="tabs">
    <ul>
        <li>tab 1</li>
        <li>tab 2</li>
    </ul>
    <div>content 1</div>
    <div>content 2</div>
</div>

CSS Markup advise:

add this if you don't want to wait until the script hide the tabs content 
.tabs > div { display: none }

Load the plugin and initialize it with the default or custom configuration

<script type="text/javascript" src="jquery.nanotab.min.js"></script>
<script type="text/javascript">
    jQuery( selector ).nanotab();
</script>

Configuration:

| type        | name     | description        | valid options         | default |
|-------------|----------|--------------------|-----------------------|---------|
|{boolean}    | _default | Default opened tab | [index|null]          |0        |
|{string}     | _effect  | Transition effect  | [showHide|slide|null] |null     |
|{string|int} | _speed   | Transition speed   | [fast|slow|ms]        |0        |

Require:

- jQuery v1.4.3+
- Custom CSS (take fiddle as example)
